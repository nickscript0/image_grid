# Item Grid
A single page app displaying a grid of items with search, tooltips, and filters. Built with ES6, Mithril.js, flowtype.org, and purecss.io.

## Build
### Load data (one time)
```bash
cd scripts
pip install -r requirements.txt
python get_isaac_items.py
```

### Run app
```bash
jspm init
sh run.sh
```
